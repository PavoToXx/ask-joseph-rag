# Predicción de Partidos de Fútbol (EPL 24/25) - ML y Power BI

Solución integral de análisis de datos y Machine Learning end-to-end diseñada para predecir resultados de partidos de fútbol de la Premier League. Abarca desde la extracción de datos hasta la visualización de predicciones en Business Intelligence.

## Objetivo funcional

Predecir el resultado de partidos de fútbol usando estadísticas de equipos y datos de jugadores. El modelo clasifica los resultados y exporta las predicciones para su análisis en un dashboard interactivo.

## Pipeline del proyecto

El flujo comienza con la extracción de datos desde Kaggle con estadísticas de equipos y jugadores de FIFA. Luego se realiza procesamiento y feature engineering en Python con limpieza iterativa, cruce de tablas y generación de features. A continuación se entrena el modelo predictivo en Jupyter Notebook para clasificar resultados. Finalmente se exportan los datos predichos a un dashboard interactivo en Power BI para analizar la precisión del modelo y estadísticas clave.

## Stack técnico

Python con pandas, scikit-learn, matplotlib y seaborn. Google Colab y Jupyter Notebook como entorno de desarrollo. Microsoft Power BI para visualización. Kaggle como fuente de datos.

## Lo que aprendí

Feature engineering con cruce de múltiples tablas de datos deportivos. Aplicación de algoritmos supervisados de clasificación. Visualización de resultados de ML en herramientas de Business Intelligence. Pipeline end-to-end desde datos crudos hasta dashboard presentable.
